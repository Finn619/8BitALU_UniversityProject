# 8BitALU_UniversityProject
This repository contains the design files and report, along with a  README for the 8-bit ALU I designed as part of my digital design course.

<img width="451" alt="image" src="https://github.com/user-attachments/assets/e4653553-9b01-44ca-a810-2608dd049629" />

This project aims to explore the implementation of one of a computer's most foundational components as part of hierarchical design for digital systems which can integrated into further combinational and sequential circuitry.
